## ✨ What is this?

Heart Flow is an interactive web experience that turns a simple apology into an emotional journey.

Instead of just showing text, it creates a flow:

Tap → Discover → Feel → Forgive 💗

---

## 💫 Features

- Multi-step interactive flow  
- Floating animated hearts background  
- Dynamic progress animation  
- Tap-to-reveal cards  
- Typewriter text effect  
- Smooth transitions using Motion  
- Playful final interaction  

---

## 🧠 Flow

Intro → Progress → Reveal → FutureHope → Final

---

## 🛠️ Tech Stack

- React  
- Tailwind CSS v4  
- Motion (motion/react + motion)  
- React Icons

## 🚀 Getting Started

```bash
git clone https://github.com/your-username/heart-flow.git
cd heart-flow
npm install
npm run dev
```

## ⚠️ Notes

- Uses motion/react, not framer-motion
- React Strict Mode may run animations twice (handled in code)
- Works best on modern browsers

## 💡 Customization

You can easily change:
-Messages (Reveal & FutureHope)
-Images
-Colors & theme
-Animation speed

## ❤️ Purpose

This project is meant to make someone feel something — not just read it.
